# TLDR.AI

Daily AI tooling breakdowns for indie hackers.

## Project Overview

This is a full-stack content platform that:
- Scrapes X/Reddit/HN for trending AI tools
- Generates deep-dive posts with code demos
- Creates Twitter threads and YouTube shorts
- Auto-posts to multiple platforms
- Monetizes via Beehiiv newsletter + sponsorships

## Tech Stack

- Astro + MDX for content
- Local LLM for writing assistance
- Playwright for browser automation
- Docker containers for isolation
- Beehiiv API for newsletter

## Quick Start

bash
npm install
npm run dev


## Content Pipeline

1. Research: Find 50 underserved sub-niches
2. Writing: Generate 1 deep-dive post + 3 Twitter threads + 1 YouTube script
3. Execution: Auto-post to X, LinkedIn, Beehiiv
4. Optimization: A/B test subject lines and convertability scores

## Monetization

- Free tier with basic content
- $29/month paid tier with exclusive tools
- Sponsorships
- Affiliate program for own AI tools
