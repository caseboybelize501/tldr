# Tool Breakdown: LangChain + OpenAI API Integration

## Overview

LangChain is a powerful framework for building applications with LLMs. This guide shows how to integrate it with OpenAI's API.

## Setup

<CodeBlock language="javascript">
{`
// Install dependencies
npm install langchain openai

// Basic setup
const { OpenAI } = require('langchain/llms/openai');

const model = new OpenAI({
  temperature: 0.9,
  openAIApiKey: process.env.OPENAI_API_KEY
});
`}
</CodeBlock>

## Usage Example

<CodeBlock language="javascript">
{`
// Simple prompt
const response = await model.call("Explain quantum computing in simple terms.");
console.log(response);

// Chain example
const chain = new LLMChain({
  llm: model,
  prompt: PromptTemplate.fromTemplate(
    "What is {subject} and why is it important?"
  )
});
`}
</CodeBlock>

## Key Features

- Easy integration with multiple LLM providers
- Chain building for complex workflows
- Memory management
- Tool calling capabilities
